# Student Exam Score Logger

## Features
- Multi-threaded student score logging
- Thread-safe file writing using synchronization
- JUnit 5 test cases

## Run Instructions
1. Compile and run Main.java.
2. Run JUnit tests
3. Output will be written to scores.csv.

   Students.java
   Scorewriter.java
   ThreadStudents.java
   MainClass.java
   ScoreTest.java
